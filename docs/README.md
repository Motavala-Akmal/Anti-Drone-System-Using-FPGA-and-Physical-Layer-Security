# Documentation

This folder contains all documentation files related to the FPGA-Based Anti-Drone Security System project.

The purpose of this directory is to keep the project explanation, technical report, presentation material, implementation notes, and supporting documents organized in one place.

---

## Folder Structure

```text
docs/
│
├── README.md
├── report/
├── presentation/
└── notes/
```

## Documentation Purpose

The documentation is prepared to explain the complete project flow from concept to implementation.

It includes:

* Anti-drone system overview
* Physical Layer Security concept
* CSI/RSSI-based detection explanation
* MATLAB simulation flow
* Verilog HDL implementation
* Vivado simulation and synthesis results
* FPGA-based risk-score generation logic
* Output analysis and project conclusion

---

## Note

This documentation is intended for academic, research, and project demonstration purposes. It helps readers understand the design logic, implementation process, simulation results, and future scope of the FPGA-Based Anti-Drone Security System.
