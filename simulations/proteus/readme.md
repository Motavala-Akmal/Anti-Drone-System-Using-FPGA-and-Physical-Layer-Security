# Proteus Simulations

This folder contains Proteus simulation files related to the FPGA-Based Anti-Drone Security System project.

The Proteus simulations are mainly used to demonstrate and analyze supporting circuit-level designs, such as power supply circuits, rectifier circuits, voltage regulation, and related hardware behavior.

## Folder Contents

```text
proteus/
│
├── README.md
├── circuit_designs/
└── outputs/
```

## Purpose

This folder is used for:

* Proteus circuit simulation files
* Power supply and rectifier circuit designs
* Diode bridge and MOSFET bridge analysis
* Voltage regulation output observation
* Circuit screenshots and waveform outputs

## Notes

Proteus files and outputs stored here support the hardware-side explanation of the project and can be used in reports, presentations, and documentation.
