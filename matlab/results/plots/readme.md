# MATLAB Plots

This folder contains plots generated from MATLAB simulations.

These plots are used to visually analyze the behavior of the Anti-Drone Security System under different scenarios.

## Purpose

The plots may show:

* Risk score variation
* Drone threat classification behavior
* CSI/RSSI-based detection results
* Eve identification outputs
* Physical Layer Security performance
* Scenario comparison results

## Notes

The images stored here can be used in the project report, presentation slides, and GitHub documentation.
