# FPGA-Based Anti-Drone Security System

## Overview

This project presents an FPGA-based Anti-Drone Security System designed for detecting, classifying, and evaluating unauthorized drone activity using Physical Layer Security concepts, wireless signal analysis, MATLAB-based modeling, and Verilog HDL implementation.

The system uses parameters such as CSI, RSSI, secrecy indicators, CAN anomaly status, authentication power, requested power, outage value, and leakage index to generate a final risk score. Based on this score, the controller classifies the threat level and supports decision-making for anti-drone response systems.

This project combines simulation, algorithm design, FPGA logic development, and hardware-oriented evaluation using MATLAB, Verilog HDL, and Xilinx Vivado.

## System Architecture

The proposed system follows a layered decision-making approach.

```text
Drone / Wireless Environment
            |
            v
CSI and RSSI Parameter Analysis
            |
            v
Physical Layer Security Evaluation
            |
            v
Risk Score Generation Logic
            |
            v
Threat Classification
            |
            v
```

The controller evaluates multiple input conditions and assigns a risk score. The final score is used to classify the drone activity as low, medium, or high risk.

## Working Principle

The system observes important wireless and security-related parameters from the drone communication environment. These parameters are processed using a risk-score-based logic.

The risk score is generated by checking different threat conditions. Each condition contributes a specific weight to the final score.

```text
Final Risk Score = Sum of Triggered Risk Weights
```

The final score is capped at 100 and is used to classify the threat level.

```text
0 - 30     : Low Risk
31 - 70    : Medium Risk
71 - 100   : High Risk
```

Based on the classification, the FPGA controller can be used to trigger suitable security responses.

## Main Input Parameters

| Parameter          | Description                                                             |
| ------------------ | ----------------------------------------------------------------------- |
| CSI                | Channel State Information used for wireless channel behavior analysis   |
| RSSI               | Received Signal Strength Indicator used for signal strength observation |
| Gamma_D            | Legitimate receiver channel/SNR-related parameter                       |
| Gamma_E            | Eavesdropper or suspicious receiver channel/SNR-related parameter       |
| Leakage Index      | Indicates possible information leakage                                  |
| Outage Value       | Represents communication failure or link degradation condition          |
| P_auth             | Authorized power level                                                  |
| P_req              | Requested power level                                                   |
| CAN Abnormal       | Indicates abnormal CAN-related behavior                                 |
| Repeated Violation | Indicates repeated suspicious activity                                  |
